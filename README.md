# YuuSlap

An interactive slap game built with Phaser 3. Click or press spacebar to slap Yuura because she's so cute and naughty :3 and so gorgeous and pretty and kind and beautiful and wise and discipline !! and cool and ethereal and majestic and amazing wow.

## How to Play

- **Click** anywhere on the screen to slap
- **Press any key** to slap
- Watch your **Score** and **APS** (Actions Per Second) in the top-left corner
- The faster you click, the faster the slap animation becomes
- Your highscore on current session will be saved as high score

## Controls

- **Mouse Click, both Right and Left** - Slap
- **Any key button** - Slap

## Play Online

[Play the game here](https://nadif243.github.io/yuu-slap/)

## Local Development

1. Clone this repository
2. Open `index.html` with a local web server (e.g., Live Server in VSCode)
3. Start slapping!
